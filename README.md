
## About 
It is made using React Native with Expo using firebase services (authentication, firestore and storage).

## Get Started ⭐️ 

```bash
# install expo cli
npm install -g expo-cli
```
``` bash
# install dependencies
npm install
```
Or use `yarn`
``` bash
yarn install
```
``` bash 
Change firebase.js to your firebase confing 
```

``` bash

# run project
expo start
```
